# Stackly - Software Services HTML Template

Professional multipurpose template for IT companies, SaaS, agencies, and consulting businesses.

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Flexbox & CSS Grid
- Fully responsive (mobile, tablet, desktop)

## Pages
- **Home:** index.html, index-2.html
- **Content:** about, services, service-details, portfolio, blog, blog-details, pricing, contact
- **Auth:** login, register
- **Admin:** dashboard, users, orders, messages, analytics
- **Utility:** 404, coming-soon, maintenance

## Features
- Dark mode toggle
- Mobile menu
- Testimonials slider
- Animated counters
- FAQ accordion
- Contact form validation

## Run
Open `index.html` in a browser. No build step required.
